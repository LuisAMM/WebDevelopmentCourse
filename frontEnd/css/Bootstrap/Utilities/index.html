Here we will show how the margin m and padding p works with bootstrap. The formula is {property}{side}-{size}. We can also use the d -display property with responsiveness d-sm-block d-none ... etc
also we can use w - width and h - height property to change the width or height of an element. The size it's decided by percentages from 25% to 100% in steps of 25% and auto :
  • property
    • m - margin
    • p - padding
  • sides
    • t - top
    • b - bottom
    • l - left
    • r - right
    • x - left & right
    • y - top & bottom
    • blank - all sides
  • size
    • 0 - set to 0
    • 1 - set to spacer*0.25
    • 2 - set to spacer*0.5
    • 3 - set to spacer*1
    • 4 - set to spacer*1.5
    • 5 - set to spacer*3
    • auto - set margin to auto
