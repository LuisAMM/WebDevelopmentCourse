The Grid System

The grid only works inside a container. To use the grid system we must use the class row on a div. Also it's really important to remenber that the grid it's divided on 12 Units

Inside a row we have columns that denotes the space that a certain element occupies

Responsive Grid

We can made the grid responsive with col-breakpoint-size. for example: col-md-8 col-sm-2 col-lg-10 col-xl-12. And they are read like this: col-md-6 -> after the md breakpoint and up occupy 6 units.. Everything below the specified breakpoint will occupy the 12 units

Responsive Images

The class img-fluid applied to the images make them compressed to enter in the container. The images must be the same height for this to work better

Grid Alignment

Text Alignment

in order to align elements in the cross axis we can use the following classes (it's applied to the row)

  • align-items-start
  • align-items-end
  • align-items-center
  • In order to align only one element we can use this classes directly on the column
    • align-self-start
    • align-self-end
    • align-self-center

Justify Content

in order to justify content on the main axis we can use the following classes (it's applied to the row)

  • justify-content-start
  • justify-content-end
  • justify-content-center
  • justify-content-between
  • justify-content-around
We can add responsiveness to our flex properties with -sm -md -lg -xl. for example: .justify-content-md-center (the same apply to align-items-sm-center)
